# KOTH API Emulator

## English

This is an API emulator for the KOTH (King of the Hill) game mode, created through reverse engineering.

It works by emulating the API environment with full debugging information for incoming and outgoing requests, along with error logging.

The database structure will not be publicly released. MySQL was used as a quick and easy solution based on available features, but you may be able to request it through personal contact (depending on my mood :D).

Everything was made in a rush. If I have time, I will refactor it.

As of May 20, 2025, the project has been functioning successfully and flawlessly for over a month.

The source code for API design was taken from Arma Reforger Tools, from a mod that is currently hidden from the general user community. If desired, it can be obtained anyway.

## Русский

Это эмулятор API для мода KOTH (King of the Hill), сделанный реверс-инжинирингом.

Работает эмулируя среду API с полной отладочной информацией приходящих и уходящих запросов, с логированием ошибок.

Структура БД не будет опубликована. Использовался MySql как решение на скорую руку по возможностям, но по личному обращению можно будет запросить (зависит от настроения :D).

Сделано всё на скорую руку, если будет время, сделаю рефакторинг.

На момент 20.05.2025 проект успешно и безотказно функционирует более месяца.

Исходники для проектирования API были взяты из Arma Reforger Tools у мода, который на данный момент скрыт от глаз общего комьюнити пользователей. При желании можно в любом случае достать.
